# Kunskapsbank

Detta är projektets kunskapsbank. ALL research som utförs ska dokumenteras här.

## OBLIGATORISKT ARBETSFLÖDE

Innan NÅGOT tekniskt beslut fattas:

1. **Hämta underlag** - Officiell dokumentation, artiklar, best practices
2. **Spara original** - Lägg filen i denna mapp (PDF, MD, eller TXT)
3. **Konvertera PDF** - `pdftotext -layout fil.pdf _txt/fil.txt`
4. **Analysera** - Skapa primer i `_analys/fil__primer.md`
5. **Referera** - Länka till underlag i beslutsdokument

## Mappstruktur

- `_txt/` - Textextrakt från PDF-filer (LÄS DESSA, inte PDF)
- `_analys/` - Primers och sammanställningar

## Primerformat

Varje primer ska innehålla:

- Källa (fullständig referens)
- Relevans för projektet
- Nyckelinsikter
- Best practices
- Påverkan på våra beslut

## VIKTIGT

- Beslut utan dokumenterat underlag är INTE tillåtna
- "Allmänt känd best practice" är INTE tillräckligt - dokumentera källan
- Vid osäkerhet: STOPPA och hämta underlag först
